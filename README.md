# Medical Image Classification XAI

Initial project structure for a medical image classification and explainable AI workflow.

## Folders

- `backend/` - API and inference code
- `frontend/` - User interface code
- `notebooks/` - Data, training, and explainability experiments
- `models/` - Saved model artifacts
- `data/` - Local datasets
- `reports/` - Results, figures, and documentation

Project scaffold created. Core logic will be added in later steps.
